# Fork et contribution GitHub

Apprendre à **copier un projet**, **travailler dessus** et **proposer ses modifications** via GitHub.

---

## Workflow

1. **Fork** : créer une copie du projet sur votre compte GitHub.
2. **Clone** : récupérer votre fork sur votre machine.
3. **Modifier et Commit** : apporter des changements et les enregistrer.
4. **Push** : envoyer vos modifications sur votre fork.
5. **Pull Request** : proposer vos modifications pour intégration dans le dépôt original.
